專案背景

大四擔任研究助理期間，參與高雄市計程車流量分析研究計畫，獨立開發資料前處理模組。

為建立計程車需求預測模型，需對原始叫車紀錄補充外部特徵變數，包含時間屬性、地點性質、道路壅塞程度與外部活動事件。

模組說明

1

平常日 / 假日判斷

PythonProject5/

透過 Abstract API 查詢台灣國定假日，判斷每筆叫車時間是否為平常日。

  • 輸入：含叫車時間欄位的 Excel
  • 輸出：新增平常日 / 假日判斷欄位
  • API：Abstract API Holidays
2

上車地點類型分類

PythonProject2/

以 Google Maps Places API 自動判斷上車地點性質，混合手動字典與 API 查詢。

類別說明
1交通樞紐
2醫療 / 教育 / 政府
3商圈 / 觀光 / 餐飲
4住宅或其他
3

最近主幹道計算

PythonProject6/

以 geopy 測地線距離演算法，找出每個上車地點的最近主幹道，串聯車流速率資料。

  • 輸入：座標清單 + 路段幾何點位
  • 輸出：最近路段名稱與距離（公尺）
  • 演算法：geodesic distance
4

活動資訊爬蟲

PythonProject3/

批次搜尋活動關鍵字，自動取得 Google 搜尋結果，內建失敗重試機制。

  • 輸入：活動查詢關鍵字清單
  • 輸出：搜尋結果網址與頁面標題
  • retry logic 處理失敗項目

環境設定

pip install pandas requests geopy beautifulsoup4 googlesearch-python openpyxl

複製 .env.example 為 .env 並填入 API 金鑰：

cp .env.example .env
